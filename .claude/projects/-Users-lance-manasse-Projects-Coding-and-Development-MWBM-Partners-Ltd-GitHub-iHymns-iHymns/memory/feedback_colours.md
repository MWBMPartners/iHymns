Use a clean, neutral, professional colour scheme — NOT bright/vivid colours.

**Why:** User explicitly rejected the amber/gold scheme and the rainbow songbook card colours, calling them "too bright", "harsh", and "like something made for kids". Wants a standardised, easy-on-the-eyes look.

**How to apply:**
- Primary: slate blue (#475569), not amber/gold
- Navbar: solid dark slate (#1e293b), no gradient
- Songbook cards: ALL the same soft grey, no rainbow
- Background: clean near-white (#fafafa), not cream
- Accents: muted teal, not orange
- Keep it professional and understated
